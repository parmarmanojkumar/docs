### Description

Noma Security is an enterprise AI security and governance platform that discovers, secures, and protects AI systems and autonomous agents across the full lifecycle. The platform delivers safety, security, and compliance controls at the runtime layer — evaluating and enforcing policy on every agent action before execution occurs.

Noma takes a holistic approach to AI security, connecting posture management, automated red teaming, and runtime protection into a single contextualized platform. Posture context informs what gets tested, test results become runtime defenses, and runtime intelligence reshapes posture priorities.

Noma was the **first company to achieve AARM Core conformance**, completing the full testing protocol against R1–R6 of the specification.

### Architecture

Noma operates as an enforcement proxy between the agent runtime and downstream tool targets. Each action request is intercepted and evaluated against a policy engine before any side effect occurs. The system maintains session-scoped context to support multi-step reasoning about accumulated intent — satisfying the AARM R3 context accumulation requirement.

The enforcement decision returns one of five outcomes defined in the AARM specification: `ALLOW`, `BLOCK`, `DEFER`, `REDACT`, or `ESCALATE`. All decisions are written to an immutable audit log before the action result is returned to the agent.

### Company

Noma Security was founded to close the enforcement gap between AI agents and the systems they interact with — a layer that traditional security tooling was never designed to protect. The company takes a holistic approach to AI security, covering autonomous agents, large language models, and RAG systems with comprehensive security and governance from a single platform.

The platform is built for organizations deploying AI agents at scale in production environments where auditability, policy control, and runtime visibility are non-negotiable. Noma has raised over $132M in funding across Series A and B, with 1,300% ARR growth and hundreds of enterprise customers across finance, life sciences, retail, and technology.
